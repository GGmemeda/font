## 一.纯函数

* 1.函数的返回结果只依赖于它的参数  例：
```bash
    const add = (a, b) => x + b;
    add(1, 2); //3
```
 add的返回结果只依赖于它的参数a和b，只要代码不变，pure(1, 2)的返回值永远是3,此为第一个条件。

* 2.函数执行过程中没有副作用

[副作用](./sideEffect.md)是指：在计算结果的过程中，系统状态的一种变化，或者与外部世界进行的可观察的交互

<font color="red">纯函数很严格，也就是说你几乎除了计算数据以外什么都不能干，计算的时候还不能依赖除了函数参数以外的数据。</font>
